# 🎯 Number Guessing Game

A simple command-line Number Guessing Game built with Python. The game generates a random number, and the player must guess it with the help of hints.

## Features

- Random number generation
- Higher/Lower hints
- Input validation
- Attempt counter
- Simple command-line interface

## Requirements

- Python 3

## Installation

```bash
git clone https://github.com/vivek235jk7/number-guessing-game.git
cd number-guessing-game
```

## Usage

Run the game using:

```bash
python number_guessing_game.py
```
